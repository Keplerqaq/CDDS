# 数据结构课程设计

## 选题

**小猫钓鱼游戏 + 世界各国制造业增加值统计分析系统**

C 语言实现，程序入口为单文件 `src/main.c`，通过菜单切换两个子系统。

## 目录结构

```
├── src/main.c                      # 全部源代码
├── src/制造业分析_inputdate.txt     # 96 国 × 21 年制造业增加值数据
├── 参考代码/            # 教材参考实现
├── 郝斌 数据结构源代码和数据结构 大纲/  # 已学数据结构源码
└── README.md
```

## 编译运行

```bash
gcc -o main src/main.c && ./main
```

## 题目一：小猫钓鱼游戏

- 链式队列存储手牌（出队头 / 入队尾）
- 链式栈存储桌面牌堆（压栈 / 出栈）
- 标记数组 O(1) 判重
- 随机洗牌发牌，两人交替出牌，自动收牌判胜负

## 题目二：制造业增加值统计分析系统

- 96 个国家 1999-2019 年制造业增加值数据
- 顺序表 + 索引数组（排序不改变原始记录顺序）
- 七项功能：导入 → 查询 → 增速计算 → 增加值排名（快排）→ 分组增速排名（选择排序）→ 统计分析 → 保存
- 四种收入等级：低收入 / 中低等收入 / 中高等收入 / 高收入
