2. 索引数组排序原理（以增加值排名为例）
──────────────────────────────────────────────────────
原始顺序：r[0]=阿尔及利亚  r[1]=阿根廷  ...  r[95]=津巴布韦
目标：按 value_added[year] 降序排，但不移动 r[]

        ┌─────────────┐         比较 r[idx_arr[i]].value_added[year]
        │  idx_arr[]  │   ──────────────────────────────────────┐
        ├─────┬───────┤                                         │
        │  0  │  15   │ ← r[15] = 中国（增加值最大）           │
        │  1  │  88   │ ← r[88] = 美国（第二大）               │
        │  2  │   5   │ ← r[5]  = 德国（第三大）               │
        │ ... │  ...  │                                         │
        │ 95  │  72   │ ← r[72] = 布隆迪（最小）               │
        └─────┴───────┘                                         │
          位置  存的                                         交换的是下标值
          (名次) 是r[]下标   ←──────────────────────────────────┘

排序后回填：
    r[15].index_va[year] = 1   (中国第1)
    r[88].index_va[year] = 2   (美国第2)
    r[72].index_va[year] = 96  (布隆迪第96)

              └─────────────────┘        └─────────────────────┘
