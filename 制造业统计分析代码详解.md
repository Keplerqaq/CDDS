# 制造业增加值统计分析系统 —— 逐行代码讲解

## 一、宏定义和数据结构

### MAXSIZE 和 YEARS

```c
#define MAXSIZE 96
#define YEARS   21          /* 1999-2019 */
```

`MAXSIZE` 是这个顺序表的最大容量，等于 96 因为数据文件里恰好 96 个国家。所有数组都用这个尺寸统一声明，不需要猜大小。后续 `RecType r[MAXSIZE]`、`int idx_arr[MAXSIZE]` 全部以它为基准。

`YEARS` 是 21 年数据（1999–2019）。用宏而不用数字：代码里要写几十次 21，万一数据扩展到 30 年，改一处就行。另外 `YEARS` 本身就是一个注释，阅读代码的人看到 `for (k = 0; k < YEARS; k++)` 立刻知道在遍历每一年。

---

### RecType —— 一条国家记录

```c
typedef struct {
    char  country[30];               /* 国家名 */
    int   country_type;              /* 0=低收入 1=中低等收入 2=中高等收入 3=高收入 */
    float value_added[YEARS];        /* 制造业增加值（亿美元） */
    float growth_rate[YEARS];        /* 增速（下标0对应1999年，仅2000-2019有效） */
    int   year[YEARS];               /* 年份标签 */
    int   index_va[YEARS];           /* 增加值排名索引（逐年） */
    int   index_gr[YEARS];           /* 增速排名索引（逐年） */
} RecType;
```

**为什么要这样设计？**

- `country[30]`：30 字节够放最长的英文名。
- `country_type`：int 存收入等级（0/1/2/3）。文本文件用中文标签更可读，代码内部用 int 更快（switch-case 而非 strcmp）。
- `value_added` 和 `growth_rate` 分开存，因为增速需要单独计算，且后续查询和分析可能只用一个。
- `index_va` 和 `index_gr` 是排名索引——每个国家每年存的是"我是第几名"。例如中国 2019 年增加值排第 2，则 `r[中国].index_va[20] = 2`。这就是教材要求的"索引数组"——排的是下标，不动原始记录，但每个记录的索引字段告诉外界"我排在第几"。
- `year[]` 存年份标签。数据文件里每年一行 21 个数，没有年份列，用这个匹配 `value_added[k]` 对应哪一年。实际上代码里直接用 `1999 + year_idx` 算，访问极少，但保留它使结构完整。

**后续用在哪**：RecType 是 `SqList.r[]` 的元素类型。整个系统的核心就是 96 个 RecType，所有功能都在读、写、算这些字段。

---

### SqList —— 顺序表

```c
typedef struct {
    RecType r[MAXSIZE];
    int     index_l[MAXSIZE];    /* 低收入国家在 r[] 中的下标 */
    int     index_ml[MAXSIZE];   /* 中低等收入 */
    int     index_mh[MAXSIZE];   /* 中高等收入 */
    int     index_h[MAXSIZE];    /* 高收入 */
    int     length;
    int     count_l, count_ml, count_mh, count_h;  /* 各组国家数 */
    int     growth_done;
} SqList;
```

**为什么顺序表选数组而不是链表？**
因为制造业统计的核心操作是：排名→随机访问任意国家的任意记录→按名次输出。数组支持 O(1) 下标寻址，链表需要 O(n) 定位。索引数组本身就是通过下标间接访问，必须用支持随机存取的结构。

**各个成员的作用和后续使用：**

- `r[]`：96 个 RecType 的连续存储。导入、查询、排名、分析全部读/写它。注意 `r[0]` 是第一个国家（0-indexed）。
- `index_l/ml/mh/h`：增速排名时用到。内部存的是某收入等级国家在 r[] 中的下标。例如 `index_l[0] = 3` 表示 `r[3]` 是某个低收入国家。在 `MVA_SqList_Sort_Gr` 中按 `country_type` 填进去，在 `MVA_SqList_Save` 中再次读取来按组写结果。
- `length`：实际导入的国家数（通常 96）。所有需要遍历全体的循环都用 `for (i = 0; i < L->length; i++)`。
- `count_l/ml/mh/h`：分组时统计各组人数。在 `MVA_SqList_Sort_Gr` 中赋值，在 `MVA_SqList_Save` 中读出形成 `sizes[]` 数组，确定每组要输出多少行。
- `growth_done`：保护标志。在 `manufacturing_system()` 里初始化为 0，`MVA_SqList_Calculate` 执行后置 1，`MVA_SqList_Sort_Gr` 和 `MVA_SqList_Analyze` 入口检查。防止跳过增速计算直接排名或分析，读到未初始化的 `growth_rate[]` 乱码。

## 二、快速排序（用于增加值排名）

### partition 函数

```c
int partition(SqList *L, int a[], int low, int high, int year) {
    int pivot = a[low];
```

- `L`：顺序表指针，用来通过 `L->r[a[i]].value_added[year]` 获取增加值。
- `a[]`：索引数组，存的是 r[] 的下标。partition 的操作对象是 `a[]` 中的值，记录数组 `r[]` 纹丝不动。
- `low`：区间左下标，`high`：区间右下标。
- `year`：在排哪一年的数据（0–20 对应 1999–2019）。
- `pivot = a[low]`：取第一个元素的下标作为基准。**注意**，这里的 pivot 是**下标**，不是增加值。后面比较时通过 `L->r[pivot].value_added[year]` 才能拿到基准值。

```c
    while (low < high) {
        while (low < high && L->r[a[high]].value_added[year] <= L->r[pivot].value_added[year])
            high--;
        a[low] = a[high];
```

**原理**：右边的小于等于 pivot？不动（因为我们要降序，小的本来就该在右边）。大于？停止，把 `a[high]` 挪到 low 位置。

`<=` 是降序排序：比 pivot 小的忽略（留在右边），比 pivot 大的才挪到左边。升序里写成 `>=`。

```c
        while (low < high && L->r[a[low]].value_added[year] >= L->r[pivot].value_added[year])
            low++;
        a[high] = a[low];
    }
```

左边的大于等于 pivot？不动（大的就该在左边）。小于？停止，挪到 high 位置。

```c
    a[low] = pivot;
    return low;
}
```

左右指针碰头后，把 pivot 放到最终位置。**返回 low**——这就是 pivot 的最终下标，递归时用它分界。

**后续用在哪**：`quick_sort_idx` 调用它做分区，`MVA_SqList_Sort_Va` 逐年调用快排。

---

### quick_sort_idx 函数

```c
void quick_sort_idx(SqList *L, int a[], int low, int high, int year) {
    if (low < high) {
        int p = partition(L, a, low, high, year);
        quick_sort_idx(L, a, low, p - 1, year);
        quick_sort_idx(L, a, p + 1, high, year);
    }
}
```

**原理**：分治法。`partition` 把数组一分为二，左边全大于 pivot，右边全小于 pivot。然后对左右递归重复。

**为什么 `low < high` 才继续？** 当区间只有一个元素（low == high）或空区间（low > high）时，无需再排，这是递归的结束条件。

**和郝斌的关系**：这和郝斌教的 `QuickSort` 完全同构。唯一的区别是郝斌直接排 `a[]` 的值，我们排的是 `idx[]`——索引数组中的下标。比较时多了一层间接访问：`L->r[a[high]].value_added[year]`。

**后续用在哪**：`MVA_SqList_Sort_Va` 中逐年调用。

## 三、七个功能

### ① 数据导入 — MVA_SqList_Read

```c
void MVA_SqList_Read(SqList *L, const char *filename) {
    FILE *fp = fopen(filename, "r");
    if (!fp) { printf("打开文件失败！请确认文件路径正确。\n"); return; }
```

`fopen(filename, "r")` 用只读模式打开。如果返回 NULL 就输出错误并返回。注意参数是 `const char *filename` 而不是硬编码路径——这让调用方可以传入不同文件名。

```c
    int n = 0;
    char temp_type[30];

    while (n < MAXSIZE) {
        if (fscanf(fp, "%s %s", L->r[n].country, temp_type) != 2) break;
```

`n` 是当前已读入的第几个国家，从 0 开始（对应 r[0]）。`temp_type` 存收入等级字符串（"低收入国家"等），读完一行就丢弃。

`fscanf` 返回成功匹配的项数。如果读不到 2 个字符串（文件结束或格式错误），`break` 退出循环。**这比 `feof` 更安全**——`feof` 只在读失败后才为真，可能导致多读一次空行。`fscanf` 提前检测避免了那个问题。

```c
        for (int i = 0; i < YEARS; i++)
            fscanf(fp, "%f", &L->r[n].value_added[i]);
```

循环 21 次，逐个数读入 `value_added[]`。`%f` 会自动跳过空格和换行。

```c
        if (strcmp(temp_type, "低收入国家") == 0)
            L->r[n].country_type = 0;
        else if (strcmp(temp_type, "中低等收入国家") == 0)
            L->r[n].country_type = 1;
        else if (strcmp(temp_type, "中高等收入国家") == 0)
            L->r[n].country_type = 2;
        else if (strcmp(temp_type, "高收入国家") == 0)
            L->r[n].country_type = 3;
```

`strcmp` 全字匹配收入等级字符串，把中文标签映射为 0/1/2/3。**为什么不用数字存？** 文本文件用中文标签更可读；代码内部用 int 更快（switch-case 而非 strcmp）。这就是"数据的外部形式"和"内部存储形式"的分离。

**后续用在哪**：这 4 个值在 `MVA_SqList_Sort_Gr` 中用 `switch (country_type)` 分组。

```c
        for (int i = 0; i < YEARS; i++)
            L->r[n].year[i] = 1999 + i;
        n++;
    }

    L->length = n;
    fclose(fp);
    printf("导入成功，共 %d 个国家。\n", L->length);
}
```

填充年份标签（1999, 2000, ...）。`n++` 表示下一个国家写入 `r[n+1]`。

`L->length = n`：记录实际读入了多少条。后面所有功能用 `for (i = 0; i < L->length; i++)` 遍历，它决定了循环次数。

**为什么最后是 `= n` 而不是 `= n - 1`？** 因为 `n` 在最后一次成功读入后 `n++`，所以 n 就是成功读入的数量。原来的 1-indexed 版本因为 `n++` 在读取之前会导致 n 多 1，才需要 `-1`。

### ② 查询 — MVA_SqList_Search

```c
void MVA_SqList_Search(SqList *L) {
    char name[30];
    int  year;

    printf("请输入国家名：");
    scanf("%s", name);
    printf("请输入年份 (1999-2019)：");
    scanf("%d", &year);

    if (year < 1999 || year > 2019) {
        printf("年份超出范围，应为 1999-2019。\n"); return;
    }
```

先读用户输入。年份范围检查避免数组越界。

```c
    int i;
    for (i = 0; i < L->length; i++)
        if (strcmp(name, L->r[i].country) == 0) break;

    if (i >= L->length) {
        printf("未找到该国。\n"); return;
    }
```

线性搜索。96 个国家很小，不需要二分查找。`strcmp == 0` 表示匹配。如果 `i >= L->length` 说明遍历了全部都没找到。

```c
    int idx = year - 1999;
    printf("%s %d年: 制造业增加值 = %.2f 亿美元",
           name, year, L->r[i].value_added[idx]);
    if (idx > 0)
        printf(", 增速 = %.2f%%", L->r[i].growth_rate[idx] * 100);
    printf("\n");
}
```

`year - 1999` 把年份转为数组下标（1999→0, 2019→20）。**为什么 `idx > 0`？** 因为 1999 年（`idx=0`）没有前值，`growth_rate[0]` 是 0 或者未初始化，显示出来没意义。`* 100` 是因为增速以小数存储（0.15 = 15%），乘 100 后 `%.2f%%` 显示。

### ③ 增速计算 — MVA_SqList_Calculate

```c
void MVA_SqList_Calculate(SqList *L) {
    for (int i = 0; i < L->length; i++) {
        L->r[i].growth_rate[0] = 0;  /* 1999 年无前值 */
```

先对所有 96 个国家，把 1999 年的增速置 0。因为公式是 `(当年–上年)/上年`，1999 没有上年。

```c
        for (int k = 1; k < YEARS; k++) {
            float prev = L->r[i].value_added[k - 1];
            if (prev == 0)
                L->r[i].growth_rate[k] = 0;
            else
                L->r[i].growth_rate[k] =
                    (L->r[i].value_added[k] - prev) / prev;
        }
    }
```

**为什么检查 `prev == 0`？** 很多穷国早期制造业数据缺失填了 0。如果 prev=0，除法会得到 `inf` 或 `nan`。规定上年为 0 则增速也记 0——这个约定来自教材。

```c
    L->growth_done = 1;
    printf("增速计算完成。\n");
}
```

**关键**：设置标志。后面 `Sort_Gr` 和 `Analyze` 先查这个，如果还是 0 就直接拒绝执行。防止用户跳过增速计算直接去排名或分析，读到未初始化的 `growth_rate[]` 乱码。

### ④ 增加值排名 — MVA_SqList_Sort_Va

```c
void MVA_SqList_Sort_Va(SqList *L) {
    int idx_arr[MAXSIZE];   /* 临时索引数组，用于快排 */
```

`idx_arr[]` 是临时索引数组，大小 96 刚好。注意：这是一个栈上的局部数组，**每次进入函数重新分配**，上次的值不会残留。

```c
    for (int year = 0; year < YEARS; year++) {
        /* 初始化索引：0, 1, ..., length-1 */
        for (int i = 0; i < L->length; i++)
            idx_arr[i] = i;
```

先把索引初始化为 `[0, 1, 2, ..., 95]`。这表示初始状态下：第 0 个位置的记录排第 0 位。

```c
        quick_sort_idx(L, idx_arr, 0, L->length - 1, year);
```

快排排序。`L->length - 1` 是最后一个元素的下标（0-indexed）。`year` 传给快排，让它比较这一年各国的增加值。排序完成后，`idx_arr[0]` 就是增加值最大的国家在 `r[]` 中的下标。

```c
        /* 将排序结果存入每个记录的 index_va */
        for (int rank = 0; rank < L->length; rank++)
            L->r[idx_arr[rank]].index_va[year] = rank + 1;
```

**关键一步**：把排序结果写回每个国家记录。`idx_arr[0]` 是第 1 名的下标 → 找到这个国家 r[idx_arr[0]]，把 `r[idx_arr[0]].index_va[year]` 设为 1（rank+1 = 名次）。名次从 1 开始记（不是 0），因为人类读第 1 名、不是第 0 名。

**后续用在哪**：这个 `index_va` 的值是后续 `MVA_SqList_Save` 输出排名的唯一数据源。保存时不再需要重新排——直接读 `index_va` 就能按名次输出。

```c
        /* 打印全部排名 */
        printf("\n%d 年增加值排名:\n", 1999 + year);
        printf("%-4s %-20s %12s\n", "名次", "国家", "增加值（亿美元）");
        for (int j = 0; j < L->length; j++) {
            int id = idx_arr[j];
            printf("%-4d %-20s %12.2f\n",
                   j + 1, L->r[id].country, L->r[id].value_added[year]);
        }
    }
}
```

打印全部排名。`j+1` 是名次，`L->r[id].value_added[year]` 从原始记录取值——完全通过索引间接访问。

### ⑤ 增速排名 — group_sort_select + MVA_SqList_Sort_Gr

#### group_sort_select（选择排序）

```c
void group_sort_select(SqList *L, int *group, int group_size,
                       int *result_idx, int year) {
    for (int i = 0; i < group_size; i++)
        result_idx[i] = group[i];
```

把组内国家的 r[] 下标拷贝到 `result_idx[]`。`group` 是预分好的下标数组（如 `index_l[]`），`group_size` 是这组有多少个国家。

```c
    /* 选择排序（降序，按增速） */
    for (int i = 0; i < group_size - 1; i++) {
        int max = i;
        for (int j = i + 1; j < group_size; j++) {
            float gr_j = L->r[result_idx[j]].growth_rate[year];
            float gr_max = L->r[result_idx[max]].growth_rate[year];
            if (gr_j > gr_max) max = j;
        }
```

选择排序的降序版。外层 i 控制第几轮，内层 j 从 i+1 到末尾找增速最大值。`> gr_max` 表示降序——谁增速大谁就放到前面。

```c
        if (max != i) {
            int tmp = result_idx[i];
            result_idx[i] = result_idx[max];
            result_idx[max] = tmp;
        }
    }
}
```

如果最大值的下标不是 i，就交换。`max == i` 说明 i 自己就是最大的，跳过交换省操作。

#### MVA_SqList_Sort_Gr

```c
void MVA_SqList_Sort_Gr(SqList *L) {
    if (!L->growth_done) { printf("请先执行增速计算！\n"); return; }
```

保护：`growth_rate` 没算好，这个函数拒绝执行。

```c
    int temp_rank[MAXSIZE];

    /* ① 将各国按收入等级分组 */
    L->count_l = L->count_ml = L->count_mh = L->count_h = 0;
    for (int i = 0; i < L->length; i++) {
        switch (L->r[i].country_type) {
            case 0: L->index_l[L->count_l++] = i; break;
            case 1: L->index_ml[L->count_ml++] = i; break;
            case 2: L->index_mh[L->count_mh++] = i; break;
            case 3: L->index_h[L->count_h++] = i; break;
        }
    }
```

**分组核心**。遍历所有国家，根据 `country_type` 把它的下标丢到对应组里。`count_l++` 是后自增：先把 i 存入 `index_l[count_l]`，然后 count_l 加 1。这样第一低收入国家在 `index_l[0]`，第二在 `index_l[1]`……

**后续用在哪**：各组下标在下面立即用来排序，而 `count_*` 的值在 `sizes[]` 中决定每组排多少行，在 `Save` 中同样用来控制输出行数。

```c
    /* ② 初始化各记录的 index_gr */
    for (int i = 0; i < L->length; i++)
        for (int year = 0; year < YEARS; year++)
            L->r[i].index_gr[year] = i;
```

初始化 `index_gr`。全部设为该国自己的下标 `i`。这是一个哨兵值——通过 `r[0].index_gr[0] == 0` 能检测"排名是否已被覆盖"，在 `Save` 中触发自动排名。后续排序会把真正的名次（1, 2, 3...）覆盖进去。

```c
    char *type_name[] = {"低收入", "中低等收入", "中高等收入", "高收入"};
    int  *groups[]     = {L->index_l, L->index_ml, L->index_mh, L->index_h};
    int   sizes[]      = {L->count_l, L->count_ml, L->count_mh, L->count_h};
```

三组并行数组。`type_name[g]`, `groups[g]`, `sizes[g]` 用同一个索引 `g`（0–3）对应同一个收入等级。这样做让循环体简洁——一个 `for (g = 0; g < 4; g++)` 搞定四组。

```c
    /* ③ 逐年分组排序并输出 */
    for (int year = 0; year < YEARS; year++) {
        for (int g = 0; g < 4; g++) {
            if (sizes[g] == 0) continue;
            group_sort_select(L, groups[g], sizes[g], temp_rank, year);
```

逐年、逐组调用选择排序。某组如果为空（`sizes[g] == 0`），跳过。

```c
            /* 将排名写回 index_gr */
            for (int rank = 0; rank < sizes[g]; rank++)
                L->r[temp_rank[rank]].index_gr[year] = rank + 1;
```

排序结果写回 `index_gr`。`rank + 1` 即组内名次（1 开头）。**注意**：`index_gr` 存的是组内排名，不是全局排名。低收入组的第 1 名、高收入组的第 1 名 都是 1。

### ⑥ 增加值分析 — MVA_SqList_Analyze

```c
void MVA_SqList_Analyze(SqList *L) {
    if (!L->growth_done) { printf("请先执行增速计算！\n"); return; }
    char name[30];
    printf("请输入要分析的国家名：");
    scanf("%s", name);

    int i;
    for (i = 0; i < L->length; i++)
        if (strcmp(name, L->r[i].country) == 0) break;

    if (i >= L->length) { printf("未找到该国。\n"); return; }

    RecType *rec = &L->r[i];
```

`rec` 是指向这个国家的指针，避免后面反复写 `L->r[i]`。

```c
    /* 遍历 21 年数据，求增加值的 min/max/和/平方和 */
    float min_va = rec->value_added[0], max_va = rec->value_added[0];
    float sum_va = 0, sum_sq_va = 0;
    float min_gr = rec->growth_rate[0], max_gr = rec->growth_rate[0];
    float sum_gr = 0, sum_sq_gr = 0;
```

初始化 min/max 为第一个值。**为什么取 value_added[0]**？因为至少有一个值（1999 年），用它做初始基准。不能用 0——如果所有增加值都大于 0（必然），min 永远是 0 就错了。

```c
    for (int k = 0; k < YEARS; k++) {
        float v = rec->value_added[k];
        if (v < min_va) min_va = v;
        if (v > max_va) max_va = v;
        sum_va += v;

        float g = rec->growth_rate[k];
        if (g < min_gr) min_gr = g;
        if (g > max_gr) max_gr = g;
        sum_gr += g;
    }

    float avg_va = sum_va / YEARS;
    float avg_gr = sum_gr / YEARS;
```

遍历 21 年，同时求最小、最大、总和。`sum / 21 = 均值`。

```c
    /* 计算方差 */
    for (int k = 0; k < YEARS; k++) {
        sum_sq_va += (rec->value_added[k] - avg_va) * (rec->value_added[k] - avg_va);
        sum_sq_gr += (rec->growth_rate[k] - avg_gr) * (rec->growth_rate[k] - avg_gr);
    }
    float var_va = sum_sq_va / (YEARS - 1);  /* 样本方差 */
    float var_gr = sum_sq_gr / (YEARS - 1);
```

方差用两步计算。**为什么除以 `YEARS-1`（20）而不是 `YEARS`（21）？** 这是**样本方差**，分母 n-1（Bessel 校正）。教材明确要求样本方差。

```c
    printf("\n%s 1999-2019 年制造业统计分析：\n", name);
    printf("  增加值 — 最小值: %.2f, 最大值: %.2f, 均值: %.2f, 方差: %.2f\n",
           min_va, max_va, avg_va, var_va);
    printf("  增速   — 最小值: %.2f%%, 最大值: %.2f%%, 均值: %.2f%%, 方差: %.6f\n",
           min_gr * 100, max_gr * 100, avg_gr * 100, var_gr);

    /* 方差解读 */
    if (var_va > 1e6)
        printf("  → 该国制造业增加值波动很大，发展不稳定。\n");
    else if (var_va < 1e3)
        printf("  → 该国制造业增加值波动较小，发展较平稳。\n");
}
```

方差解读。阈值 `1e6` 和 `1e3` 是经验值——增加值单位是亿美元，平方后方差数值级在 10⁶ 到 10⁸ 范围内，这两个阈值能大致区分"剧烈变化""中等变化""平稳发展"。

### ⑦ 保存 — MVA_SqList_Save

```c
void MVA_SqList_Save(SqList *L, const char *src_name) {
    char file_va[300], file_gr[300];
    FILE *fp;

    /* 若尚未排名，自动执行 */
    if (L->r[0].index_va[0] == 0) {
        printf("（自动执行增加值排名...）\n");
        MVA_SqList_Sort_Va(L);
    }
    if (L->r[0].index_gr[0] == 0) {
        printf("（自动执行增速排名...）\n");
        MVA_SqList_Sort_Gr(L);
    }
```

自动排名检测。`index_va[0]` 如果是 0，说明还没被排名的第 1 名覆盖（第 1 名的值是 1）。未排就自动排。

**为什么检查 r[0]？** 因为初始化时所有 `index_va` 和 `index_gr` 都是 0。只要有一个被赋值，就说明这个年份已经排过。检查第一个国家的 1999 年是个合理的哨兵（虽然不完美——理论上 r[0] 的 index_va 恰好是 0 也意味着"没排"，但 index_va 值从 1 开始，不会和 0 冲突）。

```c
    /* 生成输出文件名：去掉 .txt 后加 _Sorted / _Grouped_Sorted */
    int len = (int)strlen(src_name);
    strncpy(file_va, src_name, len - 4); file_va[len - 4] = '\0';
    strncpy(file_gr, src_name, len - 4); file_gr[len - 4] = '\0';
    strcat(file_va, "_Sorted.txt");
    strcat(file_gr, "_Grouped_Sorted.txt");
```

生成输出文件名。`len - 4` 就是去掉 `.txt`。然后拼接 `_Sorted.txt` 和 `_Grouped_Sorted.txt`。

```c
    /* 保存增加值排名（按年） */
    fp = fopen(file_va, "w");
    if (!fp) { printf("无法创建文件 %s\n", file_va); return; }
    int rank_idx[MAXSIZE];
    for (int year = 0; year < YEARS; year++) {
        /* 预建排名→国家下标映射，避免 O(n²) 扫描 */
        for (int i = 0; i < L->length; i++)
            rank_idx[L->r[i].index_va[year] - 1] = i;
```

**反向索引的精妙之处**：`index_va` 存的是"国家 → 名次"（例如 r[5].index_va[year] = 3 表示第 5 号国家排第 3）。但输出文件需要"按名次遍历"（第 1 名是谁，第 2 名是谁……）。

老方案是每找一个名次就扫描全 96 国（O(n²) 即 96×96=9216 次比较）。**现在**：`rank_idx[名次-1] = 国家下标`。一次遍历 96 国建好映射，之后 `rank_idx[0]` 就是第 1 名的国家下标（O(1) 查询）。从 O(n²) 降到 O(n)。

```c
        fprintf(fp, "\n%d 年世界各国制造业增加值排名\n", 1999 + year);
        fprintf(fp, "%-4s %-20s %12s\n", "名次", "国家", "增加值（亿美元）");
        for (int rk = 1; rk <= L->length; rk++) {
            int id = rank_idx[rk - 1];
            fprintf(fp, "%-4d %-20s %12.2f\n",
                    rk, L->r[id].country, L->r[id].value_added[year]);
        }
    }
    fclose(fp);
```

按名次 1 到 96 遍历，用反向索引瞬间找到对应国家，输出。

```c
    /* 保存增速排名（分组） */
    fp = fopen(file_gr, "w");
    if (!fp) { printf("无法创建文件 %s\n", file_gr); return; }

    char *type_name[] = {"低收入", "中低等收入", "中高等收入", "高收入"};
    int  *groups[]     = {L->index_l, L->index_ml, L->index_mh, L->index_h};
    int   sizes[]      = {L->count_l, L->count_ml, L->count_mh, L->count_h};

    for (int year = 0; year < YEARS; year++) {
        for (int g = 0; g < 4; g++) {
            if (sizes[g] == 0) continue;
            /* 预建组内排名→成员下标映射 */
            for (int j = 0; j < sizes[g]; j++) {
                int id = groups[g][j];
                rank_idx[L->r[id].index_gr[year] - 1] = id;
            }

            fprintf(fp, "\n%d 年 %s国家制造业增加值增速排名\n",
                    1999 + year, type_name[g]);
            fprintf(fp, "%-4s %-20s %10s\n", "名次", "国家", "增速");
            for (int rk = 1; rk <= sizes[g]; rk++) {
                int id = rank_idx[rk - 1];
                fprintf(fp, "%-4d %-20s %10.2f%%\n",
                        rk, L->r[id].country,
                        L->r[id].growth_rate[year] * 100);
            }
        }
    }
    fclose(fp);

    printf("排名结果已保存至：\n");
    printf("  增加值排名 → %s\n", file_va);
    printf("  增速排名   → %s\n", file_gr);
}
```

增速保存同理，按组生成反向索引。

## 六、执行流程

整个制造业系统的执行流程是线性的：

```
导入(①) → 增速计算(③) → 增加值排名(④) → 增速排名(⑤) → 分析(⑥) → 保存(⑦)
              ↓
        同时可做查询(②)
```

每个函数各司其职，通过共享的 `SqList *L` 交换数据。这就是模块化设计——每个功能独立编写和测试，通过共享数据结构协作。
